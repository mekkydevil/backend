🧪 اختبار API - نظام BNU

1️⃣ تسجيل الدخول (Login)

POST http://localhost:8001/login

2️⃣ عرض الدرجات (Get Grades)

GET http://localhost:8001/api/student/{id}/grades

3️⃣ عرض المقررات (Get Courses)

GET http://localhost:8001/api/courses

4️⃣ تسجيل مقرر (Register Course)

POST http://localhost:8001/api/courses/register

5️⃣ حساب المصروفات (Calculate Payment)

POST http://localhost:8001/api/payment/calculate

6️⃣ إحصائيات الطالب (Student Stats)

GET http://localhost:8001/api/student/{id}/stats

7️⃣ عرض الاختبارات (Get Quizzes)

GET http://localhost:8001/api/quizzes

✅ فحص حالة الخادم (Health Check)

GET http://localhost:8001/api/health
